# PayHub Admin API v1

Upload `admin.js` to:

`PayHub/backend/routes/admin.js`

## Endpoints

GET `/api/admin`

GET `/api/admin/stats`

GET `/api/admin/orders`

GET `/api/admin/traders`

GET `/api/admin/cards`

POST `/api/admin/cards/create`

PATCH `/api/admin/cards/:id/toggle`

PATCH `/api/admin/cards/:id/limits`
